About

The original log was first recorded in a Google doc until day 36. It was transfered by hand to a private wordpress blog. The blog dates were set to the date the entry covered.

The wordpress blog’s password was recovered in February 2016 and the blog’s contents were exported into wordpress xml. wp2md was used for a preliminary conversion to markdown. A bespoke python script was used to add jekyll front matter. Posts were mannually combined from one day per page to one week per page.

The site is generated using jekyll. It is styled with a tweaked version of bootstrap v4 alpha. CSS is compiled and compressed with the sass compiler. HTML is compressed with jekyll-compress-html
